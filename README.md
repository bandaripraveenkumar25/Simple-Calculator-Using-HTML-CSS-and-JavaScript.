# Simple-Calculator-Using-HTML-CSS-and-JavaScript.
Created simple calculator using HTML, CSS, and JavaScript.

Features:

✅ Basic Operations: Addition (+), Subtraction (-), Multiplication (×), Division (÷)

✅ Number Input: Digits 0-9 and decimal point support

✅ AC Button: Clear all to reset the calculator

✅ DEL Button: Delete the last entered digit

✅ Responsive Design: Works on different screen sizes

✅ Modern UI:

Gradient purple background
Color-coded buttons (numbers, operators, functions)
Smooth hover and click animations
Shadow effects for depth
✅ Error Handling: Shows "Error" message if invalid operation is attempted

How to Use:
Save the code as calculator.html
Open it in any web browser
Click buttons to perform calculations
Press "=" to see the result
Use "AC" to clear all and "DEL" to remove the last digit
The calculator uses JavaScript's eval() function to evaluate expressions, which allows you to use standard mathematical syntax. Enjoy! 🧮
